### To run locally:
Clone repository locally
```
git clone https://github.com/RaphaelBijaoui/HuffmanCoding.git
```
Compile C++ code with gcc. 
```
g++ -std=c++11 huffmancoding.cpp -o huffmancoding
```
And finally run huffmancoding executable
```
./huffmancoding
```
